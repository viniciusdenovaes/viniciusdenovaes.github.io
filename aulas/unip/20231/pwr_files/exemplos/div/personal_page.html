Meu nome é Vinicius de Novaes Pereira. Esta é a minha página pessoal. Tem algumas coisas que eu aprendi nas aulas de PWR.

Eu uso um arquivo externo para definir o estilo dos elementos da página. O nome disto é CSS. O arquivo pode ser visto no endereço daw.css.

Fiz um menu acima que fica estático enquanto a página mexe.

Também sei definir classes e id's para qualquer elemento do html.

Um elemento pode ter várias classes e vários elementos podem ter a mesma classe, como se fizessem parte de um clube.

Já a id é diferente, ela é única para cada elemento, cada elemento só pode ter uma id e cada id só pode pertencer a um elemento.

Caso eu queira descobrir se tem alguma irregularidade no meu código, eu posso verificar neste links:

  1. validador de html é uma página que valida seu código html
  2. validador de css é uma página que valida seu codigo css

Para validar o seu código nestas páginas basta entra em um dos endereços, submeter seu código, esperar alguns segundos, e a página irá dizer se tem erros ou warnings e em que linha eles estão. Bem fácil!!
