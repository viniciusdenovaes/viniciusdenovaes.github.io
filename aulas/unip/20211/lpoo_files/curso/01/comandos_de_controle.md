# Comandos de Controle

## Comando Condicional

O comando condicional `if` tem a seguinte sitaxe em Java:

* `if`:
``` java
if(condicao){
  //bloco de comandos 01
}
```

* `if-else`:
``` java
if(condicao){
  //bloco de comandos 01
} else{
  //bloco de comandos 02
}
```

* `if-else-if`:
``` java
if(condicao){
  //bloco de comandos 01
} else if(condicao){
  // bloco de comandos 02
}
```
