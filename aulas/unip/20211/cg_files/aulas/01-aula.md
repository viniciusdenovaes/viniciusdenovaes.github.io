
### Transformações Afins

* Translação
* Escala
* Rotação
* Cisalhamento

## Matriz do Ponto

```
p = [ x  y ]
```

## Translação
Transladar no eixo x com um valor de T_x e no eixo y com um valor de T_y

```
M_T = [T_x  T_y]
```

```
p' = p + M_T
```

## Escala
Aumentar o desenho em S_x em relação ao eixo x e S_y em relação ao eixo y

```
M_E =
[S_x  0  ]
[  0  S_y]
```

```
p' = p * M_E
```
