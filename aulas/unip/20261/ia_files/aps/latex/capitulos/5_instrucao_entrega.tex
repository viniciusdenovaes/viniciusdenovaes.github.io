\chapter*{Instruções para o Desenvolvimento e Entrega}
\addcontentsline{toc}{chapter}{Instruções para o Desenvolvimento}
\begin{itemize}
    \item O trabalho deve ser realizado em grupo de até 5 alunos;
    \item Cada grupo deverá apresentar oralmente o projeto.
    \item A entrega do trabalho escrito deverá ser realizada exclusivamente pelo sistema de trabalhos acadêmicos da UNIP, disponível em:\url{https://trabalhosacademicos.unip.br/entrega/}.
	\item A entrega do trabalho deverá ser feita até a Data-limite para a entrega ou postagem dos trabalhos acompanhados das fichas de registro das Atividades Acadêmicas (APS, AC, EC, TC e Extensão) e resolução de exercícios (ED) no sistema acadêmico, pelos alunos. Disponível em: \url{https://www.unip.br/servicos/aluno/manual/tradicional/calendarios.aspx}.
\end{itemize}
