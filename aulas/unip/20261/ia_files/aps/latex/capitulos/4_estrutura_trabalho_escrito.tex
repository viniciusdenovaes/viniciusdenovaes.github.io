\chapter{Estrutura do Trabalho Escrito}

O trabalho escrito deverá conter:

\begin{itemize}
    \item Capa;
    \item Introdução;
    \item Descrição do problema tratado;
    \item Descrição das coluções propostas;
    \item Descrição das técnicas utilizadas;
    \item Prints das telas desenvolvidas;
    \item Conclusão;
\end{itemize}
