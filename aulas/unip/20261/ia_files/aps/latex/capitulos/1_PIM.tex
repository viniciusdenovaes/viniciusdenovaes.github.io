\chapter{Projeto e Desenvolvimento}

O Projeto e Desenvolvimento tem como finalidade proporcionar ao aluno a aplicação prática dos conhecimentos adquiridos durante o curso, promovendo a integração entre teoria e prática.

Neste semestre, o projeto será desenvolvido com foco na elaboração de um algoritmo e programa para resolver o TSP, um problema clássico NP-Completo.

O objetivo é entender que uma solução ótima do TSP, para uma aplicação do mundo real, não é viável. Sendo preciso procurar uma solução alternativa
que não seja ótima.
