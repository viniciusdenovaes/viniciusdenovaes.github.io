
# Projeto de Matemática Aplicada e Desenvolvimento Android (NP1 - 2026)
O objetivo central deste trabalho é o desenvolvimento de um aplicativo Android funcional que utilize a Matemática Aplicada para resolver um problema prático do cotidiano

### Requisito Matemático Fundamental
O núcleo do aplicativo deve ser a execução de lógica matemática baseada em dados reais. Para isso, o sistema deve, obrigatoriamente 

##### receber uma entrada numérica: 
A interface deve conter pelo menos um campo de entrada onde o usuário insira um valor numérico

##### Processar um cálculo: 
O aplicativo deve realizar, no mínimo, um cálculo matemático processando a entrada fornecida pelo usuário

##### Exibir o resultado: 
O valor processado deve ser apresentado de forma clara na interface de saída

#### Exemplos Práticos:
Exemplos de problemas matemáticos sugeridos:
 - Cálculo de Índice de Massa Corporal (IMC)
 - Conversão de unidades de medida (ex: peso, distância ou temperatura)
 - Conversão de moedas com taxas de câmbio

### Documentação e Explicação Matemática
Além do código, o grupo deve entregar um projeto escrito (uma folha, frente e verso) que detalhe a fundamentação teórica do trabalho.

Nesta parte, deve-se dar ênfase a:

##### Descrição do Problema: 
Explicar a relevância e a aplicação prática do problema matemático escolhido

##### Demonstração do Cálculo: 
Detalhar a fórmula ou lógica matemática utilizada para converter a entrada do usuário no resultado final

##### Interface e Usabilidade: 
Explicar sucintamente como o design (IHC) facilita a interação do usuário com a operação matemática (clareza na entrada e precisão na saída)
